**Step indicator** — stepper bar ที่ header dialog พร้อม filled/active/inactive states

| Step | หน้าตา |
|---|---|
| 1 Upload | Drag & drop zone + `<input type="file" accept=".xlsx,.csv">` จริง + Download Template button |
| 2 Map Columns | Preview 5 rows จาก "Excel" + highlight column ตามที่เลือก + dropdowns map BIB column / Identifier column |

- **Conflicts**: card warning สีเหลือง พร้อม radio buttons เลือก runner ที่จะ keep BIB
- **Unmatched**: list rows แสดง BIB + name + identifier + row number (skipped)
